# ORIEL Consolidated Repository Status (0.1.0–0.8.1)

This repository consolidates the historical ORIEL milestones from 0.1.0 through 0.7.3 with the active implementations delivered for 0.7.4 through 0.8.1.

## Included milestone history

- 0.1.0 Initial console interpreter
- 0.2.0 Collections, loops, files and JSON
- 0.3.0 Diagnostics, typing and package foundations
- 0.4.0 Modules, language intelligence and API preview
- 0.5.0 Database preview and OpenAPI
- 0.6.0 Application services, validation and authentication utilities
- 0.7.0 Initial console foundation
- 0.7.1 Core audit and stabilization
- 0.7.2 Static type-system completion
- 0.7.3 Modules and standard library
- 0.7.4 Package manager
- 0.7.5 Language server and VS Code intelligence
- 0.8.0 Compiler, IR, bytecode and virtual machine
- 0.8.1 Debugger, profiling and optimization

## Active repository version

The active package version is **0.8.1**. Earlier milestones are represented by the source evolution, changelog, roadmap, verification records, examples and implementation documentation retained in this single repository.

## Release rule

A milestone is considered releasable only when source code, automated tests, documentation, CI verification and tagged artifacts all correspond to the same commit.
