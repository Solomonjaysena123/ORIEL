
# Oriel CLI 0.1

## Install

```bash
python -m pip install .
```

Development installation:

```bash
python -m pip install -e .
```

## Commands

```bash
oriel version
oriel run examples/hello.orl
oriel check examples/hello.orl
oriel new my_project
```

## Expected output

```text
Oriel 0.1.0
Hello AryaLabs
Welcome to the Oriel Software Language
```
